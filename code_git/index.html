================================================================================================================================================================================================ documentation pour toi zinc 😁😁 pour apprendre git ================================================================================================================================================================================================ 1. naviguer et se mettre à l’intérieur du répertoire du projet 2. cd.. pour remonter 3. cd nomDuDossier pour descendre 4. ls ou dir pour afficher la liste du contenu 5. git init pour que git initialise ce dossier en projet git. Cela ajoute les trois arbres nécessaire au suivi de version: 6. le répertoire de travail (la version actuelle): working directory 7. l’index : la liste des choses qui sont préparées (la commande pour ajouter au « stage » est « add« ) 8. le head qui correspond a ce qui a été « commit« , c’est à dire vraiment envoyé comme version 9. git add . est la commande qui « stage » tous les documents, qui les ajoute à l’index. Pour info, on peut sortir de la pile un document avec la commande $ git reset HEAD votreFichier 10. git commit -m "mon premier commit" permet de transférer les fichiers en attente dans l’index, vers le Head. Le message permet de savoir à quoi cela correspond. 11. Créez le projet sur Github.com, en ne mettant PAS de Readme. Il vaut mieux l’ajouter après, une fois que les fichiers ont été uploader, pour éviter tout conflit. 12. Copier le lien vers le répertoire, situé généralement en haut du projet dans une boite bien visible: le lien se termine par « .git » 13. git remote add origin leLienCopié va désigner le répertoire distant comme cible du projet. 14. Répondez aux information soit dans la boite de dialogue qui s’affiche, soit dans la suite de la commande. 15. git remote -v J’avoue que je n’ai pas vraiment compris, il me semble que c’est juste la vérification que les choses fonctionnent dans les deux sens. 16. git push origin master est la commande qui va finalement envoyer les fichiers ================================================================================================================================================================================================ bientôt la suite 😁😁😁😁😁😁 ================================================================================================================================================================================================ jnsdfinsdiunsdiufniudsf wdvpk,wdokg,dgpk dwgkwdgpkd,g wdgwjndgodsg, wgjnkgnoj dgpkwjndijndswoj
